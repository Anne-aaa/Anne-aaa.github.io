第五章 格式

  1. 垂直方向上的区隔：
    • 封包声明、导入声明和每个函数间都有空白行隔开。 每个空白行是一条线索，标示出新的独立概念。
    • 被调用的函数应该放在执行调用的函数下面
  2. 横向格式：
    • 空格字符： 乘法因子间没有空格，因为乘法具有较高优先级。加减法用空格隔开，因为优先级低。
      • 赋值操作符周围加上空格，达到强调目的。
      • 函数调用括号中的参数一一隔开，表示参数是互相分离的。
